Verify if UEFI mode is enabled.
# ls /sys/firmware/efi/efivars
If the directory does not exist, the system may be booted in BIOS mode.


Verify network connection.
# ping archlinux.org


Update the system clock.
# timedatectl set-ntp true
# timedatectl status


Partition the disks.
# fdisk -l
# cfdisk
Mount point   Partition	  Partition type ID	  Boot flag	  Suggested size
/	            /dev/sda1	  83: Linux	          Yes	        23â€“32 GiB
[SWAP]	      /dev/sda2	  82: Linux swap	    No	        More than 512 MiB


Format the partitions.
# mkfs.ext4 /dev/sda1
# mkswap /dev/sda2
# swapon /dev/sda2


Mount the file systems.
# mount /dev/sda1 /mnt


Select the mirrors.
# vim /etc/pacman.d/mirrorlist
Move the best mirror to the top of the list.


Install the base packages.
# pacstrap /mnt base


Generate an fstab file.
# genfstab -U /mnt >> /mnt/etc/fstab
Check the resulting file.
# vim /mnt/etc/fstab


Change root into the new system.
# arch-chroot /mnt


Set the time zone.
# ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
# hwclock --systohc


Localization.
# vi /etc/locale.gen
Uncomment:
en_US.UTF-8 UTF-8
zh_CN.UTF-8  UTF8
# locale-gen
Create the locale.conf file, and set the LANG variable accordingly.
# vi /etc/locale.conf
LANG=en_US.UTF-8


Create the hostname file.
# vi /etc/hostname
myhostname


Add matching entries to hosts.
# vi /etc/hosts
127.0.0.1	localhost
::1		localhost
127.0.1.1	myhostname.localdomain	myhostname


Set the root password.
# passwd


Boot loader.
# pacman -S grub os-prober
# grub-install --target=i386-pc /dev/sda
# grub-mkconfig -o /boot/grub/grub.cfg


Exit the chroot environmen.
# exit


Reboot.
# reboot

Log in as root.
